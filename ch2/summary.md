# ch 2 환경 설정하기

### godoc

* 패키지문서를 볼 수 있다.
* `godoc` 명령을 이용하여 패키지 문서를 확인할 수 있다.
* 파이썬의 `help()` 명령어랑 비슷한 것으로보인다.
* 웹사이트는 https://godoc.org 로 가면된다.

### Oracle

* 따로 설치가 필요하다.
* 메뉴얼은 여기 https://docs.google.com/document/d/1SLk36YRjjMgKqe490mSRzOPYEDe0Y_WQNRv-EiFYUyw/view

### Vet

* `go tool vet`
* 소스 코드 검사툴

### Fix

* `go tool fix`
* 변경된 API를 자동으로 변경해준다.

### Test

* `go test`
* 테스트 진행하는 기본 툴이다.
